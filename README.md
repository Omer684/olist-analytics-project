# 🛒 Olist E-Commerce Analytics: Python, SQL, & Power BI

## 📌 Project Overview
This project is an end-to-end data analytics workflow using a real-world Brazilian e-commerce dataset containing over 100,000 orders. The goal was to analyze customer purchasing behavior, identify revenue trends, segment customers, and visualize key performance indicators (KPIs).

## 🧰 Tech Stack
* **Python:** `pandas`, `matplotlib`, `seaborn` (Data cleaning, Cohort Analysis)
* **SQL:** MySQL (CTEs, Window Functions, RFM Segmentation)
* **Data Visualization:** Power BI (DAX, Data Modeling, UI Design)

## 📊 Key Business Findings
1. **Total Revenue:** The platform generated **$13.59 Million** across delivered orders.
2. **Geographic Dominance:** Customers in São Paulo (SP) dominate the platform, generating nearly 40% of all revenue.
3. **Acquisition vs. Retention:** Repeat customers make up only **~3%** of the user base. Month-2 retention is **<1%**, indicating a heavily acquisition-driven business model.
4. **Seasonality:** Revenue saw a massive spike in **November 2017 (~$988K)**, consistent with Black Friday.
